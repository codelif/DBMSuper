# DBMSuper
Sem III DBMS Lab PBL

## Instructions:

- Download and configure a MySQL database
- Get its credentials (host, user, pswd, db);
- run `SOURCE shop.sql;` in the database (don't forget `USE db;`) 
- add the credentials in `app.py`
- in venv run `pip install -r requirements.txt`
- `fastapi dev`
